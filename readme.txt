Technology used: PHP
Server used: Xampp(v3.2.2)
The app folder called is divided into 3 parts:
1- Images folder: "img" ;it containes all the pictures used in the app
2- PHP and HTML files for each page
3- CSS files
4- Database file "applicationshop.sql"

-To begin,you have to import the database file "application_shop" into your phpmyadmin databases
-Now you can start!
-To acces to the app,go to the browser and enter this adress: http://localhost:8012/app_shop/
(don't forget to change the port name)
